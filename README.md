**Open PS2 Loader Remix** is a personal fork of the original
[Open PS2 Loader (OPL)](https://github.com/ps2homebrew/Open-PS2-Loader).

This project focuses on customization, visual tweaks, and personal
experiments while keeping full compatibility with the original OPL.

>[!NOTE]
This is **NOT an official version** of Open PS2 Loader.

This Project is currently in progress

##  Goals of this fork

- Visual customization (themes, UI tweaks)
- Personal workflow improvements
- Learning and experimentation with OPL internals

No guarantees of stability are provided.
